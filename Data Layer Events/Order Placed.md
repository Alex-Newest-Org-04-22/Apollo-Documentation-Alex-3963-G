# Order Placed

### 

## Javascript Code
```js
window.dataLayer = window.dataLayer || [];
dataLayer.push({ ecommerce: null });  // Clear the previous ecommerce object.
dataLayer.push({
  "event": "Order Placed",
    "ecommerce": {
        "alternate_pickup_person": "<alternate_pickup_person>",
        "currency": "<currency>",
        "items": [
            {
                "item_id": "<item_id>",
                "item_name": "<item_name>"
            }
        ],
        "transaction_id": "<transaction_id>",
        "value": <value>
    }
});
```
